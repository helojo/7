    这是一道特别特别简单的C语言入门题，让初学者明白如何运用C语言头文件以及输入输出。

    题目描述

    Calculate a+b

    输入

    Two integer a,b (0<=a,b<=10)

    输出

    Output a+b

    样例输入

    1 2

    样例输出

    3
    1 #include<stdio.h>   
2 int main()
3 {
4     int a,b;
5     scanf("%d %d",&a,&b);
6     printf("%d",a+b);
7     return 0;
8     
9 }
