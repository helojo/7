    fopen()函数

    函数作用

    用来打开一个文件

    头文件

    #include <stdio.h>

    用法

    FILE *fopen(char *filename, *type);

    TYPES

    "r"——only read

    "w"——only write

    "a"——无文件时创建一个文件

    "r+"——可读可写

    "w+"——可读可写

    "a+"——在文件的开始进行初始化，附在文件的末尾

    fclose()函数

    作用

    用来关闭一个由fopen()函数打开的文件

    int fclose(FILE *stream);

    返回值

    成功：return 0;

    失败：非0值

     

    fprintf()函数

    作用

    向文件中写入

    用法

    int fprintf(FILE *stream, char *format, <variable-list>);

    example

    fprintf(fd, "%d ,%s", 100, YES);

    返回值

    成功：返回写入的字符数

    失败：返回一个负数
