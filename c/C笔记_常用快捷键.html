    1、第一部分

    Ctrl + up/down              以光标所在行为中心上下移动文本；
Ctrl + left/right           左右跳过一个单词或符号；
Ctrl + end                  跳至文本末尾；
Ctrl + delete               删除光标后的单词或符号；
Ctrl + enter                在光标所在行插入空白行；
Ctrl + Shift + enter        在光标所在行的下一行插入空白行；

Alt + right                 字母拓展；
Alt + up/down               上下移动光标所在行；

Shift up/down/left/right    上下左右选择
Shift end                   选中光标至行尾内容；
Shift delete                删除光标所在行；

    2、第二部分

    Ctrl + Shift + N            新建项目
Ctrl + Shift + A            添加新项
Shift + Alt + A             添加现有项
F5                          开始调试
Ctrl + F5                   开始执行
F9                          切换断点
Ctrl + B                    设置条件断点
F10                         逐过程
F11                         逐语句
Ctrl + G                    转到行
Ctrl + Shift + T            转到文件
Ctrl + U                    转换为小写
Ctrl + Shift + U            转换为大写
Ctrl + K,Ctrl + F           自动格式化代码
Ctrl + K,Ctrl + C           注释代码
Ctrl + K,Ctrl + U           取消注释代码
