    命令解析器---解析输入命令并执行操作

    • 命令或目录补齐快捷键：TAB键

     

    • 遍历史 输入：
      1. 　　小键盘：上 或 CTRL+P
      2.        小键盘：下 或 CTRL+N

     

    • 光标移动：
      1.        小键盘：左 或 CTRL+B
      2.        小键盘：右 或 CTRL+F
      3. 　　CTRL+A：移动到头部
      4. 　　CTRL+E：移动到尾部

     

    • 字符删除：
      1. 　　CTRL+H 或 退格键： 删除光标   前   面的字符
      2. 　　CTRL+D 或 DELETE键 ： 删除光标   后  面的字符
      3. 　　CTRL+U：删除光标  前  面的字符串
      4. 　　CTRL+K：删除光标  后  面的字符串
