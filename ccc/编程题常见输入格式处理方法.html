    1、声明数组长度和数组内容

    案例：

    输入样例:
6
-2 11 -4 13 -5 -2

    处理方式：

    //数据定义
int count = 0;
int* inter;

//获取数据
cin >> count;
inter = new int[count];
for (int i = 0; i < count; i++)
{
    int a = 0;
    cin >> a;
    inter[i] = a;
}

    2，声明最大长度

    案例：

    数字数量 N < 20，输入数字的最大值小于 256.

输入：
10 10 11 12 12 11 16

    处理方式：

    //数据定义
int count = 0;//数组有效长度
int* inter  = new int[20];

//获取数据
int a = 0;
int i = 0;
while (cin >>a)
{
    inter[i++] = a;
    count++;
}

//开始处理

    备注：

    说明1：当读到输入结束时，cin >> 返回 0，循环也结束。

说明2：在调试程序时，键盘输入的数据，用CTRL-Z组合作为输入结束，即键盘输入设备的“文件末尾”。
