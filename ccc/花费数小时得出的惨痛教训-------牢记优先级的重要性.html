    以上代码第一个if后面的内容永远被执行。

    这里需要考虑到优先级问题，一般而言,算数运算>关系运算>逻辑运算>位运算>赋值运算。

    上面代码中，m==1返回false,false与3进行"或"运算,3被视为true。结果返回true.同理最后返回true，所以if后的语句就一直被执行了。

    常见的还有下面这种情况

    if (1< 4 &  4< 3)
{
cout << "if 被执行了"；
}

    　　

    同级运算符,从左往右依次计算；1<4返回true ，true与3比较，相当于1与3比较，结果仍为true.

    可以这样改写

    if (1< 4 & 4 < 3)
{
cout << "if 被执行了"；
}

    反思：还是受到自然语言的影响太大了，往往不会注意到这些细节。牢记运算的优先级，注意细节,准程序员的计算机之路还很漫长。
