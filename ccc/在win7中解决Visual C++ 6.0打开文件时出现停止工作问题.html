    • 在win7中解决Visual C++ 6.0打开文件时出现停止工作问题
    原文出处:本文由博客园博主箫燕轩剑韵提供。
    原文连接:https://www.cnblogs.com/jianshneg/p/11254059.html

    在使用Visual C++ 6.0打开文件时可能会出现下面的情况

    这可能是Vc6.0和win7兼容性问题。

    方法：

    下载filetool即可

    链接：https://pan.baidu.com/s/1Xmx0XI0Dy9uZGJEQW4cHQg
    提取码：drgz
    下载之后，解压到一个目录，我这个是解压到桌面所形成的，找到如下图标

    点击，解压到一个目录（注意记住这个目录不要找不到了）

    找到这个目录，在该目录中，用Vc6.0打开tool.dsw文件，

    之后在编译一下，

    便出现如下情况

    将debug里的filetool.dll文件放到C:\Program Files (x86)\Microsoft Visual Studio\Common\MSDev98\AddIns

    也可是其它目录关键是看你这个Vc安装在哪里找到这个就行：Microsoft Visual Studio\Common\MSDev98\AddIns

    如下debug目录：

     然后打开Vc6.0找到:

    工具->定制->附加项和宏文件

    勾选

    点击A O图标的O图标把它放到打开文件的那一个地方就可以了（其实可以拖到你想要放的地方就可以了）

    点击后即可以打开文件了
