    • C++中typedef enum 和 enum
    原文出处:本文由博客园博主zhouleyuan提供。
    原文连接:https://www.cnblogs.com/zhouleyuan/p/10751598.html

    在C++中，这两种定义枚举类型的关键字用法和效果相同，推荐使用前者。typedef enum多用在C语言中。

    在C语言中，如果使用typedef enum定义一个枚举类型，比如：

    typedef enum TokenType
{
    blah1 = 0x00000000,
    blah2 = 0X01000000,
    blah3 = 0X02000000
} MyType;

    则可以通过 TokenType MyType 使用所定义的枚举类型。如果使用enum定义一个枚举类型，比如：

    enum TokenType
{
    blah1 = 0x00000000,
    blah2 = 0x01000000,
    blah3 = 0x02000000
}MyType;

    则需要通过 enum TokenType MyType 使用所定义的枚举类型。
