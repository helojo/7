    # c++primer 61页

     

    typedef char* ptr

const ptr cstr=0;

     

    cstr 是一个指向 char 的常量指针，

    一种错误的理解是将语句等同于const char* ptr cstr;

    但 ptr 实际上是指向 char 的指针

     

    正确的理解应是
    const ptr -->常量指针类型c-p
    c-p cstr 声明一个c-p类型的变量cstr，其为常量指针cstr

     

     

    而通过上一个语句认识到typedef与指针组合的含义，下一个语句也很好理解了

    const ptr *ps;

     

    声明符从右往左结合，

    首先ps是一个指针变量，其次其指向const ptr类型的变量（常量指针类型，该类型指向char ）。
