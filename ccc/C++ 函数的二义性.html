    函数二义性是一种编译时发生的错误，造成二义性调用的原因

    是在函数匹配时两个或多个函数提供的匹配一样好，编译时找

    不到唯一的最佳匹配。

    列：

     1 #include <iostream>
 2 using namespace std;
 3 int fun(int a, int b = 23) {
 4     return a + b;
 5 }
 6 int fun(int a) {
 7     return a * a;
 8 }
 9 int main() {
10     int a = 34;
11     cout << fun(a);
12     return 0;
13 }

    在此程序中，fun函数出现二义性，

    出现有多个 重载函数 "fun" 实例与参数列表匹配
