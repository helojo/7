    赋值运算符重载框架如下：

     1 ClassName& ClassName::operator = (ClassName obj)
 2 {
 3     if (this != &obj)
 4     {
 5         delete obj;
 6         使用new为调用者对象分配形参obj对象同样大小的动态存储空间;
 7         将形参obj对象的动态存储空间中的数据赋给调用者对象
 8     }
 9     return *this;　　　　//返回给调用者
10 }
