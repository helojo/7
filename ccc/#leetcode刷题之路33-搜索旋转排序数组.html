    • #leetcode刷题之路33-搜索旋转排序数组

    假设按照升序排序的数组在预先未知的某个点上进行了旋转。
    ( 例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2] )。
    搜索一个给定的目标值，如果数组中存在这个目标值，则返回它的索引，否则返回 -1 。
    你可以假设数组中不存在重复的元素。
    你的算法时间复杂度必须是 O(log n) 级别。

    示例 1:
    输入: nums = [4,5,6,7,0,1,2], target = 0
    输出: 4
    示例 2:
    输入: nums = [4,5,6,7,0,1,2], target = 3
    输出: -1

     

    思路：分情况讨论

    #include <iostream>
#include <vector>
using  namespace std;
int search(vector<int>& nums, int target) {
    if(nums.size()==0) return -1;
    if(nums.size()==1&&nums[0]==target) return 0;
    if(nums.size()==1&&nums[0]!=target) return -1;
    int low=0;
    int high=nums.size()-1;
    while(low<=high)//二分法
    {
        int mid=(low+high)/2;
        if(nums[mid]==target) return mid;
        if(nums[mid]>=nums[low])//mid的左侧为单增序列
        {
            if(target>=nums[low]&&target<nums[mid]) high=mid-1;//target在单增序列上
            else low=mid+1;//target不在单增序列上
        }
        else//mid的右侧为单增序列
        {
            if(nums[mid]<target&&nums[high]>=target) low=mid+1;//target在单增序列上
            else high=mid-1;//target不在单增序列上
        }
    }
    return -1;
}

int main() {
    int target=1;
    vector<int> a={3,1};
    int ans=search(a,target);
    std::cout <<ans<< std::endl;
    return 0;
}
