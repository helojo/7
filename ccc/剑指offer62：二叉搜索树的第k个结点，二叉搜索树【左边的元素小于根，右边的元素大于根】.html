    1 题目描述

    　　给定一棵二叉搜索树，请找出其中的第k小的结点。例如， （5，3，7，2，4，6，8）    中，按结点数值大小顺序第三小结点的值为4。

    2 思路和方法

    　　二叉搜索树【左边的元素小于根，右边的元素大于根】，按照中序遍历的顺序打印出来正好就是排序好的顺序。 中序遍历这颗二叉树，然后设置一个变量，访问一个变量的时候就加一，判断这个变量和k是否相等，如果相等，则将当前的这个结点返回即可

    3 C++核心代码

     1 class Solution {
 2 public:
 3     TreeNode* KthNode(TreeNode* pRoot, int k) {
 4         if (k < 0 || pRoot == nullptr) {
 5             return nullptr;
 6         }
 7         N = k;
 8         InOrder (pRoot);
 9         return res;
10     }
11     void InOrder (TreeNode* r) {
12         if (r == nullptr || tag) {
13             return;
14         }
15         InOrder (r->left);
16         ++n;
17         if (n == N) {
18             res = r;
19             tag = true;
20             return;
21         }
22         InOrder (r->right);
23     }
24     
25     int N {0};
26     int n {0};
27     bool tag {false};  // 找到的标记，提前结束没必要的搜索
28     TreeNode* res {nullptr};
29 };
