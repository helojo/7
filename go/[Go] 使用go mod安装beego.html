    • [Go] 使用go mod安装beego
    原文出处:本文由博客园博主陶士涵提供。
    原文连接:https://www.cnblogs.com/taoshihan/p/11884638.html

    需要go升级到1.12或以上

    mkdir gomodtest
cd gomodtest
go mod init gomodtest

    创建一个server.go

    package main
import "github.com/astaxie/beego"
func main() {
    beego.Run()
}
    go run server.go
自动查找依赖并且下载依赖的包到$GOPATH/pkg/mod/github.com/astaxie
