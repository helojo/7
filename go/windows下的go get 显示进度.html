    • windows下的go get 显示进度
    原文出处:本文由博客园博主feiquan提供。
    原文连接:https://www.cnblogs.com/feiquan/p/11423836.html

    我的Go版本是：go1.12.7

    1.在你的Go安装目录下找到

    D:\Go\src\github.com\tools\godep\vendor\golang.org\x\tools\go\vcs\vsc.go

    2.编辑Git命令让其显示进度

    CreateCmd:   "clone --progress {repo} {dir}",

    3. 重定向输出为标准输出

    cmd.Stdout = os.Stdout    // 重定向标准输出
cmd.Stderr = os.Stderr    // 重定向标准输出

     4.进行重新编译

    可以参考：https://studygolang.com/articles/7643
